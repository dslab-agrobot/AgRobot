This directory is used for storing all photos and videos taken during the development of agrobot. Before you put them, please note:

1. All photos and videos are categorized by date. Before you put them, check the date, if there exists a subdirectory for the date, put them into there, otherwise, create the directory for that date with name "YYYY-MM-DD".

2. The name of the image shall follow the format of "IMG-YYYYMMDD-I", here 'I' indicates the index of the photo.

3. The name of the video shall follow the format of "VIDEO-YYYYMMDD-I", here 'I' indicates the index of the photo.

4. Directory "2019-4-30" is an example for these requirements, please check it.
