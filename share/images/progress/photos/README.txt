This directory is used for storing all photos taken during the development of agrobot. Before you put photos, please note:

1. All photos are categorized by date. Before you put photo, check the date, if there exists a subdirectory for the date, put photo into there, otherwise, create the directory for that date with name "YYYY-MM-DD".

2. The name of the image shall follow the format of "IMG-YYYYMMDD-I", here 'I' indicates the index of the photo.

3. Directory "2019-4-30" is an example for these requirements, please check it.
